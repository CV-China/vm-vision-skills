# 场景分类决策树

> 定位引导场景的5级精细决策树，从客户描述逐步定位到具体方案。

## 决策树使用方法

1. 从级别1开始，根据客户描述确定分支
2. 逐级向下，每级回答一个关键问题
3. 到达叶节点后，输出对应的完整方案
4. 如果客户信息不足以判断，主动询问缺失的关键信息

---

## 级别1：相机数量

### 单相机

→ 进入级别2（安装方式）

**特征**：只有一个相机，拍摄物料或料盘

### 双相机（上下各一）

→ 进入级别2（安装方式）

**特征**：上相机+下相机，通常上相机拍料盘/目标，下相机拍物料/对象

### 双相机（同侧两个）

→ 进入级别2（标定条件）

**特征**：两个相机安装在同一侧，通常用于大物料拍摄对角

---

## 级别2：安装方式 / 标定条件

### 对于单相机

**静止安装（Eye-to-Hand）**
- 相机固定不动，机构移动物料
- 标定时：标定板放在机构上移动
- → 进入级别3

**运动安装（Eye-in-Hand）**
- 相机安装在机械手上，随机械手移动
- 标定时：标定板固定不动，机械手带动相机移动
- → 进入级别3

### 对于双相机（上下）

**双相机静止**
- 上下相机都固定安装
- 标定方式：映射标定 + 下相机N点标定
- → 进入级别3

**混合安装**
- 一个相机静止，一个相机运动（通常上相机随R轴旋转）
- 标定方式：需额外考虑旋转坐标系转换
- → 进入级别3

### 对于双相机（同侧）

**有大标定板**
- 标定方式：标定板标定（大标定板同时出现在两个相机视野中）
- 统一两相机坐标系后，按单相机方案处理
- → 进入级别3

**有小标定板**
- 标定方式：映射标定（小标定板在两相机间移动）
- → 进入级别3

**无标定板**
- 标定方式：双十二点标定（机构分别和两个相机做十二点标定）
- 注意：两相机第五点偏移量需补偿
- → 进入级别3

---

## 级别3：应用类型

### 抓取（Pick）

**目标**：从工作台/料盘抓取物料放到目标位置

**核心流程**：相机拍物料 → 计算抓取坐标 → 机械手执行抓取

→ 进入级别4（有无旋转）

### 纠偏（Align）

**目标**：机械手已吸取物料，纠正物料姿态偏差后放置

**核心流程**：相机拍已吸取的物料 → 计算纠偏量 → 机械手纠偏放置

→ 进入级别4（有无旋转）

### 对位（Match）

**目标**：物料与目标位置精确对位贴合（如FPC贴合、屏幕组装）

**核心流程**：上相机拍目标+下相机拍物料 → 计算对位偏差 → 执行对位

→ 进入级别4（对位方式）

---

## 级别4：旋转 / 对位方式

### 对于抓取场景

**无旋转（仅XY平移，不需要输出角度偏差）**
- 机构：XY平台，无R轴，或有R轴但不需要角度补偿
- → 进入级别5（共轴/偏心）

**有旋转（需要输出角度偏差，R轴旋转补偿）**
- 机构：XYR一体或分离轴，需要角度纠正
- 需要十二点标定（含旋转中心标定）
- → 旋转抓取方案

### 对于纠偏场景

**无旋转（不需要输出角度偏差）**
- 仅纠正XY位置偏差
- → 进入级别5（共轴/偏心）

**有旋转（需要输出角度偏差）**
- 需纠正XY+角度偏差
- 需要十二点标定（含旋转中心标定）
- → 旋转纠偏方案

### 对于对位场景

**映射对位**
- 上相机拍目标，下相机拍物料，通过映射标定统一坐标系
- → 映射对位方案

**各自基准对位**
- 上下相机各自与机构标定，通过机构坐标系统一
- → 双十二点对位方案

**偏心对位**
- 物料特征点与贴合点有固定偏移
- → 偏心对位方案（需建立工件坐标系）

---

## 级别5：共轴 / 偏心

### 共轴（工具中心 = 旋转中心）

**特征**：吸嘴/夹爪中心与R轴旋转中心重合

**方案**：
- 标定：九点标定（仅需平移旋转标定）
- 计算模型：单点抓取 / 单点纠偏
- VM模块：平移旋转标定 + 单点抓取/纠偏

### 不共轴（偏心）

**特征**：吸嘴/夹爪中心与R轴旋转中心不重合

**方案**：
- 标定：十二点标定（需额外标定旋转中心，VM模块选"平移旋转标定"）
- 计算模型：单点抓取 / 单点纠偏
- VM模块：平移旋转标定 + 单点抓取/纠偏

## 分离轴场景特殊处理

当R轴与XY轴分离时（分离轴场景），需额外处理：

### 分离轴判断标准
- R轴挂Y轴：R轴安装在Y轴滑块上
- R轴挂X轴：R轴安装在X轴滑块上
- R轴独立：R轴完全独立于XY轴
- 完全分离：XYR三轴完全独立

### 标定五原则
1. **主相机决定原则**：主相机决定标定方式
2. **符号原则**：根据机构运动方向确定符号
3. **取反原则**：某些场景需取反坐标
4. **分离原则**：与R轴分离的才是分离
5. **吸放旋转标定修正**：吸放时旋转中心可能变化

### 计算四原则
1. **对象原则**：R轴抓着的才是对象
2. **拍照位决定原则**：拍照位决定坐标系
3. **抓取成立原则**：抓取时坐标系必须一致
