# VM关键操作提示

> VisionMaster平台定位引导的关键操作提示，以流程图为主、操作要点为辅。重引导、重操作，轻原理、轻长篇大论。

---

## 一、特征提取流水线（核心）

**所有定位引导方案的图像处理部分都遵循此流水线**：

```
┌───────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐
│  图像采集   │───→│  粗定位     │───→│  精定位     │───→│  抓边输出   │
│ (相机拍照)  │    │(高精度匹配) │    │(圆/线+修正) │    │(坐标+角度) │
└───────────┘    └────────────┘    └────────────┘    └────────────┘
                      │                  │                  │
                      ▼                  ▼                  ▼
               定位ROI区域         提取亚像素特征       输出精确坐标
               修正偏移量          位置修正ROI          和旋转角度
```

### 1.1 粗定位

| 项目 | 内容 |
|------|------|
| **VM模块** | 高精度匹配 / 模板匹配 |
| **目的** | 在图像中定位目标区域，为精定位提供ROI偏移修正 |
| **关键参数** | 匹配分数阈值≥0.7；金字塔层级3-5；角度范围覆盖物料旋转范围 |
| **模板选择** | 选择物料上稳定、唯一的特征区域作为模板 |
| **输出** | 匹配位置(X,Y) + 匹配角度(R) + 匹配分数 |

**粗定位操作要点**：
1. 框选模板时避开毛刺、划痕等不稳定特征
2. 角度范围设置需覆盖物料最大旋转角（如±10°设为±15°）
3. 匹配分数阈值不宜过高（>0.9会漏检），也不宜过低（<0.5会误检）
4. 多物料时用多个粗定位，分别框选不同特征

### 1.2 精定位

| 项目 | 内容 |
|------|------|
| **VM模块** | 圆查找+位置修正 / 直线查找+位置修正 |
| **目的** | 在粗定位修正后的ROI内，提取亚像素级精确特征点 |
| **关键参数** | 边缘过渡带<3像素；极性选择（黑到白/白到黑）；搜索方向 |
| **位置修正** | 必须连接粗定位输出，用粗定位结果修正精定位ROI位置 |

**精定位操作要点**：
1. **位置修正模块必须加**：粗定位→位置修正→精定位，确保ROI不跑偏
2. 圆查找：选择完整圆孔，避免半圆（拟合精度差）
3. 直线查找：选择长直边，避免短边和弧线段
4. 边缘极性必须与实际图像一致（反了会抓错边）
5. ROI范围不宜过大（噪声多）也不宜过小（跑出ROI）

**精定位→抓边搭配**：

| 特征类型 | 精定位模块 | 抓边方式 | 适用场景 |
|---------|-----------|---------|---------|
| 圆孔 | 圆查找 | 输出圆心坐标 | IC芯片定位孔、螺丝孔 |
| 直边 | 直线查找 | 输出线段+角度 | 产品边缘、定位边 |
| 两直线交点 | 2×直线查找 | 计算交点 | 矩形物料角点 |
| 圆+直线 | 圆查找+直线查找 | 圆心+角度 | 有孔有边的产品 |

### 1.3 抓边

| 项目 | 内容 |
|------|------|
| **目的** | 从精定位结果中提取最终坐标和角度，输出给标定/生产计算模块 |
| **输出** | 特征点坐标(X,Y) + 角度(R) |
| **稳定性要求** | 同一特征连续30次抓取，坐标波动≤1像素，角度波动≤0.1度 |

**抓边稳定性验证**：
1. 固定物料连续拍照30次，检查输出坐标波动
2. 波动>1像素：检查光源稳定性、边缘质量、参数设置
3. 角度波动>0.1度：检查直线长度是否足够、边缘是否清晰

## 二、标定模块操作提示

### 2.1 平移旋转标定（N点标定）

**VM模块路径**：定位引导 → 平移旋转标定

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  放置标定板    │───→│ 9点平移采集   │───→│ 3点旋转采集   │
│ (视野中心)    │    │(Z字形等间距)  │    │(原点+2个角度) │
└──────────────┘    └──────────────┘    └──────────────┘
                                              │
                                              ▼
                                    ┌──────────────┐
                                    │  查看标定结果  │
                                    │(状态/精度/一致性)│
                                    └──────────────┘
```

**关键参数**：

| 参数 | 典型值 | 注意事项 |
|------|--------|---------|
| 标定类型 | 共轴→平移标定；不共轴→平移旋转标定 | 不确定时选平移旋转标定 |
| 相机模式 | 相机运动 / 相机静止 | 根据安装方式选择 |
| 自由度 | 缩放+旋转+纵横比+倾斜+平移 | 一般不改 |
| 平移次数 | 9次 | 覆盖视野1/2以上 |
| 旋转次数 | 3次 | 角度≥15度，建议20-30度 |

**验证标准**：

| 检查项 | 合格标准 |
|--------|---------|
| 标定状态 | 1=成功 |
| 平移像素精度 | < 1像素 |
| 旋转像素精度 | < 0.5像素 |
| 旋转一致性 | 1=一致（-1需验证） |
| 手性一致性 | 1=一致（-1需验证） |

### 2.2 旋转标定

**VM模块路径**：定位引导 → 旋转标定

**关键参数**：

| 参数 | 典型值 | 注意事项 |
|------|--------|---------|
| 旋转角度 | 15-30度 | 必须≥15度 |
| 旋转点数 | 3点 | 原点+两个旋转位置 |

**验证**：旋转中心偏差<2像素；角度越大拟合越准

### 2.3 映射标定

**VM模块路径**：定位引导 → 映射标定

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ 标定板放机构上 │───→│ 上相机拍照    │───→│ 移动到下相机   │
│              │    │ 提取特征点    │    │ 下相机拍照     │
└──────────────┘    └──────────────┘    └──────────────┘
                                              │
                                              ▼
                                    ┌──────────────┐
                                    │ 映射标定模块   │
                                    │ 求映射矩阵     │
                                    └──────────────┘
```

**关键约束**：小标定板映射时禁止旋转；R轴不能旋转；放下标定板后不能再移动

## 三、生产计算模块操作提示

### 3.1 单点抓取

**VM模块路径**：定位引导 → 单点抓取

```
┌──────────┐    ┌──────────────┐    ┌──────────────┐
│ 移动到    │───→│ 拍照+特征提取  │───→│ 计算偏差      │
│ 拍照位    │    │(粗→精→抓边)   │    │(当前-基准)    │
└──────────┘    └──────────────┘    └──────────────┘
                                          │
                                          ▼
                                ┌──────────────┐
                                │ 输出抓取坐标   │
                                │(基准+物理偏差) │
                                └──────────────┘
```

**关键参数**：

| 参数 | 说明 | 注意事项 |
|------|------|---------|
| 拍照位 | = 标定第五点 | 不一致需加偏移量 |
| 基准像素点 | 示教时记录 | 更换物料需重新示教 |
| 基准物理点 | 示教时抓取坐标 | 机械手基准位置 |

### 3.2 单点纠偏

**VM模块路径**：定位引导 → 单点纠偏

```
┌──────────┐    ┌──────────────┐    ┌──────────────┐
│ 吸取物料  │───→│ 移动到拍照位  │───→│ 拍照+特征提取  │
│ (基准姿态) │    │              │    │(粗→精→抓边)   │
└──────────┘    └──────────────┘    └──────────────┘
                                          │
                                          ▼
                                ┌──────────────┐
                                │ 输出纠偏坐标   │
                                │(基准-物理偏差) │
                                └──────────────┘
```

**与抓取的区别**：纠偏时物料在机械手上，输出=基准物理点-物理偏差（注意减号）

### 3.2 单点映射对位

**VM模块路径**：定位引导 → 单点映射对位

```
┌──────────────┐    ┌──────────────┐
│ 下相机拍物料   │───→│ 上相机拍料盘  │
│(粗→精→抓边)   │    │(粗→精→抓边)   │
└──────────────┘    └──────────────┘
        │                  │
        ▼                  ▼
┌──────────────────────────────────┐
│ 映射标定转换 → 计算对位偏差 → 输出 │
└──────────────────────────────────┘
```

**关键参数**：映射标定文件 + 工件坐标系（偏心对位时需建立）

                                    ▼            ▼

### 4.4 标定验证

**VM工具路径**：定位引导 → 标定验证

| 检测项 | 标准值 | 不合格修正 |
|--------|--------|-----------|
| 平移像素精度 | < 1像素 | 检查标定点提取精度 |
| 旋转像素精度 | < 0.5像素 | 检查旋转中心标定 |
| 物理平移精度 | < 0.02mm | 检查机构精度和标定步长 |
| 物理旋转精度 | < 0.05度 | 检查R轴精度和旋转标定角度 |
| XY轴夹角 | 89.8°-90.2° | 检查标定点分布和机构正交性 |

## 五、常见VM操作注意事项

### 标定操作
- 拍照位必须与标定第五点一致，否则需加拍照位偏移量
- 标定步长建议10-30像素，太小放大噪声，太大降低局部精度
- 旋转标定角度必须≥15度，否则旋转中心拟合不准
- 运动相机标定时，标定板固定不动；静止相机标定时，标定板放在机构上移动

### 示教操作
- 示教时必须保持与生产时相同的姿态
- 基准像素点在示教时记录，生产时用于计算偏差
- 每次更换物料或调整机构后需重新示教

### 生产操作
- 生产时拍照位必须与标定/示教时一致
- 工作距离（相机到物料距离）必须保持一致
- 光源稳定性直接影响重复性

### 分离轴特殊处理
- R轴抓着的才是"对象"
- 分离轴场景需根据16种场景符号表确定X/Y/R的符号（1或-1）
- 详见 `assets/1.标定原理/分离轴场景总结与兼容方法.md`
