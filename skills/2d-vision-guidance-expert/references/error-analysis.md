# 定位误差分析参考

> 定位引导项目误差排查和精度提升的核心参考，包含排查流程、测试方法、CheckList和VM诊断工具使用。

---

## 1. 四步排查流程

定位问题排查应严格按照以下顺序逐级进行，不可跳步：

| 步骤 | 排查阶段 | 排查重点 |
|------|---------|---------|
| **第一步** | 机构及成像排查 | 机构精度和稳定性、成像质量是否满足精度要求 |
| **第二步** | 标定过程排查 | 标定数据正确性、标定过程稳定性、标定结果误差分析 |
| **第三步** | 示教过程排查 | 示教数据是否正确、示教选择的特征是否稳定 |
| **第四步** | 生产过程排查 | 生产拍照位是否正确、生产时机构是否稳定、生产时特征是否一致 |

### 1.1 机构及成像排查

**机构规范性**：相机与成像平面必须垂直

**成像质量**：
- 对焦清晰，边缘过渡带不超过3个像素
- 检查是否存在透视或桶形/枕形畸变（直线边不应变弯曲）

### 1.2 标定过程排查

**标定参数准确性**：检查基本参数（相机模式、自由度等）是否填写正确。自由度一般建议选择第二个（缩放、旋转、纵横比、倾斜及平移）

**标定参数合规性**：
- 九点轨迹覆盖视野1/2以上
- 标定角度不小于5度，条件满足时大于30度

**标定数据质量**：
- 单张图像特征点偏差不超过1个像素
- 不同图像间特征点差异不超过1个像素
- 标定轨迹X/Y轴垂直，轨迹稳定无波动
- 九点数据等差（像素差控制在1个像素以内），趋势正确（同增/同减）
- 物理坐标与机械手偏移坐标对应，旋转数据角度与实际对应

**标定结果**：
- 标定状态：1成功 / 0失败
- 平移像素精度 < 1像素
- 旋转像素精度 < 0.5像素
- 旋转一致性：1一致 / -1相反
- 手性一致性：1相同 / -1相反

**标定文件分析**（使用 calib_analyzer 脚本）：
- 调用 `${SKILL_DIR}/scripts/calib_analyzer.py` 的 `analyze_calib_file(xml_path, camera_mount)` 或 `analyze_calib_content(xml_str, camera_mount)` 函数分析（`camera_mount` 取 `'stationary'` 静止/`'moving'` 运动）
- 输出 12 项检测指标（去掉标定状态和标定误差状态），含平移像素精度、旋转像素精度、步长极差、角度极差、物理平移/旋转精度、旋转一致性（XML与计算值对比）、左右手一致性（XML与计算值对比）、平移像素最大误差（前9点拟合仿射对这9个平移点的重投影误差最大值）、XY轴夹角、旋转中心偏差
- 每项含 pass/fail 判定、标准值、排查方向
- 分析结果通过 `{{CALIB_REPORT}}` 嵌入诊断 HTML 的独立卡片

### 1.3 示教过程排查

- 示教位物理坐标为工具移动到刚好能抓取物料时的机械手坐标，确认记录正确
- 基准图像（示教后保持物料不动，移动相机到第五点拍照位采集）及基准图像点记录正确

### 1.4 生产过程排查

- 拍照位坐标与标定第五点拍照位一致（可产生X/Y偏移），但拍照姿态（相机旋转角度）必须保持不变
- 焦距必须与标定时一致
- 平移偏差排查：只输出moveoffsetX/Y，看工具中心能否移动到物料中心
- 旋转一致性验证：给定补偿角度，若抓取姿态正确则旋转一致性正确
- 手性一致性判断：机械手坐标系与图像坐标系手性一致为1，不一致为-1
- 旋转中心拟合准确性：将旋转后图像特征绕旋转中心旋转回基准姿态，与第五张做差，差值越小拟合越准

## 2. 动静态测试方法与标准

### 2.1 静态测试

| 项目 | 内容 |
|------|------|
| **测试方法** | 保持拍照位不动，连续100次拍摄同一特征 |
| **数据输出** | 通过VM输出Excel结果 |
| **评估指标** | 波段峰峰值在1个像素以内为正常 |
| **目的** | 测试相机精度及机构是否存在抖动，排查硬件稳定性 |

### 2.2 动态平移测试

| 项目 | 内容 |
|------|------|
| **测试方法** | 机构分别沿X/Y正负方向移动回拍照位拍照，连续往返100次拍摄同一特征 |
| **数据输出** | 通过VM输出Excel结果 |
| **评估指标** | 机构运动误差小于0.02mm为正常 |
| **目的** | 测试机构动态走位复位精度，排查机构运动精度 |

### 2.3 动态旋转测试

| 项目 | 测试方法 | 标准 |
|------|---------|------|
| **角度变化测试** | 机械手分别沿正反方向旋转一定角度（与标定时角度相当） | 图像特征角度变化在0.1度以内 |
| **复位精度测试** | 回到拍照位查看复位精度 | 回拍照位图像特征点变化在1个像素以内 |

## 3. 标定排查检查项与判定标准

| 序号 | 检查项 | 判定标准 |
|------|--------|---------|
| 1 | 标定状态 | 1表示成功 |
| 2 | 标定误差状态 | 0表示正常 |
| 3 | 平移像素精度 | < 1像素 |
| 4 | 旋转像素精度 | < 0.5像素 |
| 5 | 平移像素步长极差 | < 1像素 |
| 6 | 平移物理角度极差 | < 0.01度 |
| 7 | 平移图像角度极差 | < 0.01度 |
| 8 | 物理平移精度 | < 0.02mm |
| 9 | 物理旋转精度 | < 0.05度 |
| 10 | 角度旋转一致性 | 1一致 / -1相反|
| 11 | 坐标系左右手一致性 | 1一致 / -1相反|
| 12 | 平移像素最大误差 | < 1像素 |
| 13 | 旋转像素最大误差 | < 1像素 |
| 14 | 物理XY轴夹角 | 89.8度 - 90.2度 |
| 15 | 旋转中心像素偏差度 | < 1像素 |

### 标定数据质量速查

- 九点图像数据等差：像素差控制在1个像素以内
- 九点物理间距是否等间距
- 标定路径是否为直线
- X精度与Y精度是否相同
- 第五点和第11点是否重合
- 脉冲当量设置是否正确

## 4. 6种常见问题及排查方式

| 序号 | 问题现象 | 可能原因 | 排查方式 | 标准 |
|------|---------|---------|---------|------|
| 1 | 定位不准确 | 运动机构不稳定 | 对每个相机做动静态测试：静态结果大=视觉稳定性差；动态结果大=整体稳定性差 | 静态<1pix, 动态<0.02mm |
| 2 | 特征提取不稳定 | 成像质量不好 | 查看成像特征边缘过渡带 | 边缘过渡带<3像素 |
| 3 | 标定矩阵存在误差 | 标定时提取的图像数据不稳定 | 检查标定数据 | 每次特征点稳定在1像素以内 |
| 4 | 定位不准 | 拍照位角度姿态与标定时不一致 | 比对标定时的拍照姿态与生产时的拍照姿态 | 二者角度必须一致 |
| 5 | 定位不准 | 机构拍照未走到对应位置 | 比对同一点位的生产图片，观察背景是否偏移 | 背景无偏移 |
| 6 | 引导误差偏大 | 吸嘴末端与物料平台不水平，吸放侧压 | 使用水平尺测试吸嘴平面与目标平面是否平行 | 两者平行 |

## 5. 八大类CheckList检查项

### 5.1 视觉硬件

| 检查项 | 内容 |
|--------|------|
| 光源 | 控制可靠、打光效果满足要求、吸嘴不影响成像、准备多种光源方案 |
| 相机 | 选用成像质量好的相机（非仅看分辨率）、被测物体占视野2/3、安装垂直牢固 |

### 5.2 结构硬件

| 类别 | 内容 |
|------|------|
| 重复精度 | 完成静态/动态精度测试、R轴精度验证、输出精度报告 |
| 硬件设计 | 垂直度/水平度/平面度检查、避免极限位置、旋转臂长时建立工具坐标系 |
| 运动控制 | 到位确认后再拍照、拍照后加延时、垂直离开物体表面 |

### 5.3 产品分析

确认产品厚度影响、尺寸一致性、尺寸公差、基准边、边对应关系，输出产品分析报告。

### 5.4 标定

- 畸变矫正：评估畸变系数，确认是否需要矫正
- N点标定：范围2/3视野、保存图像和点位、点位精度1像素、角度精度0.1度、等间距/直线路径/X=Y精度/第5点和第11点重合/脉冲当量正确
- 映射标定：上下相机图像四点对应正确

### 5.5 定位粗调

大角度大位移下：中心对中心验证通过、偏移值方式验证通过、证明计算方法正确。

### 5.6 定位精调

- 补偿值：一致性高时才补偿、补偿值不超过0.1
- 抓边稳定性：大量物料图片测试、保证特征点稳定

### 5.7 复检

- 尽量在一张图片内测量
- 否则保证光源一致性

### 5.8 VM方案

- 格式化前接条件检测、关键模块OK才输出
- 修改模板后固定基准点、ROI正确
- 输出X/Y/R坐标及抓边特征渲染图

## 6. VM异常诊断工具5步排查流程

```
静态测试 → 动态测试 → 标定验证 → 生产验证 → 排查列表
```

| 步骤 | 功能 | 排查目标 | 数据输入 |
|------|------|---------|---------|
| 1. 静态测试 | 测试相机精度及机构是否存在抖动 | 硬件稳定性 | CSV文件（100次静态拍摄） |
| 2. 动态测试 | 测试机构动态走位复位精度 | 机构运动精度 | CSV文件（100次往返拍摄） |
| 3. 标定验证 | 校验标定数据合理性 | 标定准确性 | N点标定文件 |
| 4. 生产验证 | 排查生产步骤是否正确 | 流程正确性 | 生产流程配置 |
| 5. 排查列表 | 其他注意事项全面排查 | 潜在问题 | - |

**运行环境**：Win10+ 64位 / VisionMaster V4.3.0+ / 基础版加密狗

## 8. 快速排查指南

| 问题现象 | 优先检查页面 | 关键检测项 |
|---------|------------|-----------|
| **重复性差** | 静态测试页 | 点位散点图、波段峰峰值 |
| **走位不准** | 动态测试页 | 复位精度、运动误差 |
| **系统性偏差** | 标定验证页 | 旋转一致性、手性一致性 |
| **某些位置不准** | 生产验证页 | 拍照位、示教点 |
| **精度不达标** | 排查列表页 | 全面系统排查 |
