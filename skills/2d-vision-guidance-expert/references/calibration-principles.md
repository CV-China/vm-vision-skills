# 标定原理参考

> 2D定位引导标定的核心参考，以操作决策为主，精简原理说明。覆盖6种标定分类、九点/十二点选择、映射标定、分离轴处理和VM标定模块配置。

---

## 1. 六种标定分类速查

| 编号 | 标定类型 | 场景 | 相机模式 | 物理坐标处理 | VM模块 |
|------|---------|------|---------|-------------|--------|
| 1 | 静止九点 | 相机固定，机构带物料走9位 | 相机静止 | 相对第5点坐标 | N点标定，选"相机静止" |
| 2 | 运动九点 | 相机随机构运动，物料固定 | 相机运动 | 物理坐标取负号 | N点标定，选"相机运动" |
| 3 | 静止十二点 | 相机固定，机构有旋转自由度 | 相机静止 | 前9点平移+后3点旋转 | 平移旋转标定 |
| 4 | 运动十二点 | 相机随机构运动，有旋转自由度 | 相机运动 | 根据分离轴原则取反+转绝对 | 平移旋转标定 |
| 5 | 静止映射 | 两个相机都静止，标定板共用 | - | 分别求M1和M2 | 标定板标定模块 |
| 6 | 移动映射 | 两个相机，物料在两相机间移动 | - | 记录两次机械坐标 | 映射标定(有标定板)/相机映射模块（使用有四个不共线角点的物料） |

### 标定选择决策

```

                    │ 有几个相机？     │
                    └────┬───────┬────┘
                    单相机       双相机
                     │              │

              ┌────────────┐  ┌──────────────┐
              │ 有旋转自由度？│  │ 两相机都静止？│
              └──┬─────┬──┘  └──┬───────┬───┘
                无     有       是       否
                │      │       │        │
                ▼      ▼       ▼        ▼
             九点标定  十二点   静止映射  移动映射
             (共轴)  (不共轴)  (标定板)  (映射标定/相机映射)
```

## 2. 九点标定 vs 十二点标定

### 选择依据：共轴 vs 不共轴

| 维度 | 九点标定 | 十二点标定 |
|------|---------|-----------|
| **本质** | 平移标定（不带旋转中心） | 平移旋转标定（九点+旋转标定） |
| **点数** | 9个点 | 12个点（9平移+3旋转） |
| **适用** | 共轴（吸嘴与旋转轴重合） | 不共轴（吸嘴与旋转轴分离） |
| **旋转中心** | 不需要 | 需要，归一化后原点在旋转中心 |

**选择原则**：
- 共轴（机械手建立工具坐标系，旋转绕抓取点）：九点标定
- 不共轴（机械手无需建立工具坐标系，旋转绕旋转中心）：十二点标定
- 不确定时选十二点标定（适用绝大多数场合）

## 3. 映射标定选择

```
┌──────────────────────────┐
│ 两个相机都静止 + 标定板可共用？│
└──────────┬───────────────┘
          是 │              否
             ▼                ▼
       ┌──────────┐    ┌──────────────┐
       │ 静止映射  │    │ 物料需在两相机│
       │(标定板标定)│    │ 间传递？      │
       └──────────┘    └──┬───────┬──┘
                       是       否
                       │        │
                       ▼        ▼
                 ┌──────────┐ ┌──────────────┐
                 │ 移动映射  │ │ 双十二点标定  │
                 │(映射模块) │ │(各自与机构标定)│
                 └──────────┘ └──────────────┘
```

**映射标定注意事项**：
- 静止映射：标定片不能平移/旋转后再拍第二个相机
- 移动映射：R轴不能旋转；放下标定片后不能再移动；四点物料映射时上下相机成镜面关系，四点顺序是反的
- 优先使用标定片而非物料

## 4. 分离轴处理

### 分离轴判断标准

```
R轴挂Y轴：R轴安装在Y轴滑块上
R轴挂X轴：R轴安装在X轴滑块上
R轴独立：R轴完全独立于XY轴
完全分离：XYR三轴完全独立
```

### 标定五原则

| 原则 | 内容 | 操作要点 |
|------|------|---------|
| 1. 主相机决定 | 做旋转标定的是主相机 | 上下不重要，看谁做旋转标定 |
| 2. 符号原则 | 取反符号只在标定物理坐标上体现 | 计算部分用常规模块 |
| 3. 取反原则 | 分离则取反、运动则取反、运动吸放则取反 | 取反只用在标定物理坐标上 |
| 4. 分离原则 | 与R轴分离的才是分离 | R轴决定"谁是被旋转的" |
| 5. 吸放修正 | 相机和R轴相对静止时需修正矩阵第三列 | 非分离轴：吸放位-拍照位；分离轴：拍照位-吸放位 |

**重要**：取反符号只用在标定时的物理坐标上，计算时填的所有拍照位都用机械臂原始值，不带符号。

### 计算原则

| 原则 | 内容 |
|------|------|
| 1. 对象原则 | R轴抓着的才是对象；生产拍的不是R轴上抓着的=抓取，拍的是R轴上抓着的=纠偏 |

### 16种分离轴场景符号汇总表

| 编号 | 场景描述 | X | Y | R |
|------|---------|---|---|---|
| 1 | 常规轴_相机静止 | 1 | 1 | 1 |
| 2 | 常规轴_相机随XYR运动 | -1 | -1 | -1 |
| 3 | 常规轴_相机随XY运动不随R运动 | -1 | -1 | 1 |
| 4 | R轴与X轴分离_相机静止 | -1 | 1 | 1 |
| 5 | R轴与Y轴分离_相机静止 | 1 | -1 | 1 |
| 6 | R轴与X轴分离_相机随YR运动 | -1 | -1 | -1 |
| 7 | R轴与Y轴分离_相机随XR运动 | -1 | -1 | -1 |
| 8 | R轴与X轴分离_相机只随Y运动 | 1 | -1 | 1 |
| 9 | R轴与Y轴分离_相机只随X运动 | -1 | 1 | 1 |
| 10 | R轴与X轴分离_相机只随X运动 | -1 | 1 | 1 |
| 11 | R轴与Y轴分离_相机只随Y运动 | 1 | -1 | 1 |
| 12 | R轴与XY分离_相机静止 | -1 | -1 | 1 |
| 13 | R轴与XY分离_相机随R运动 | 1 | 1 | -1 |
| 14 | R轴与XY分离_相机随XY运动 | -1 | -1 | 1 |
| 15 | R轴与XY分离_相机只随X运动 | -1 | 1 | 1 |
| 16 | R轴与XY分离_相机只随Y运动 | 1 | -1 | 1 |

**规律**：相机装在R轴上→3个-1；1=不取反，-1=取反

## 5. VM4.2标定模块关键参数

### 平移旋转标定模块

| 参数 | 选项 | 说明 |
|------|------|------|
| 标定类型 | 平移标定 / 平移旋转标定 | 共轴选平移，不共轴选平移旋转 |
| 标定点获取 | 触发获取 / 手动获取 | 采集方式 |
| 相机模式 | 相机运动 / 相机静止 | 根据安装方式选择 |
| 自由度 | 3种组合可选 | 默认：缩放+旋转+纵横比+倾斜+平移 |
| 平移次数 | 默认9次 | 覆盖视野1/2以上 |
| 旋转次数 | 默认3次 | 角度≥15度 |
| 组合标定 | 开启/关闭 | 小视野旋转后平移回视野内标定 |

### 旋转标定模块

| 参数 | 说明 |
|------|------|
| 输入方式 | 按点、按坐标 |
| 图像点 | 特征点的像素坐标 |
| 物理旋转角度 | 机构每次旋转的角度值 |
| 旋转次数 | 最少三次 |

### 定位模块

| 模块 | 关键参数 | 限制条件 |
|------|---------|---------|
| 单点抓取 | 标定类型、像素点、示教物理点、拍照位、标定文件 | 不支持多相机联立、分离轴、拍照位变化 |
| 单点纠偏 | 同上 | 不支持多相机联立、分离轴、拍照位变化 |
| 单点映射对位 | 对象/目标像素点、示教物理点、平移旋转标定文件、映射文件 | 不支持单相机循环对位 |

### 设置要点

1. 标定顺序：约定走S形，通讯协议标定则轨迹任意
2. 自由度选择：默认（缩放+旋转+纵横比+倾斜+平移），一般不需要改
3. 刷新信号：填1会自动加载最新的标定文件
4. 示教：传入示教物理值，将当前图像特征点作为基准
5. 生产拍照位：最好与标定第5点拍照位一致

## 6. 标定常见问题速查

| 问题 | 排查方法 | 判定标准 |
|------|---------|---------|
| 拍照位角度姿态不一致 | 比对标定/生产拍照姿态 | 角度必须一致 |
| 机构未走到对应位置 | 比对同一点位生产图片 | 背景无偏移 |
| 运动机构不稳定 | 静态100次+动态100次测试 | 静态<1pix，动态<0.02mm |
| 吸嘴与平台不水平 | 水平尺测试 | 两者平行 |
| 大角度偏移不稳定 | 检查机械手各关节位置 | 无受限 |

**定位应用三部曲**：标定 → 示教 → 做差
