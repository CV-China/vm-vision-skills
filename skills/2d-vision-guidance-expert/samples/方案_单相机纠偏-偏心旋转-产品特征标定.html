🎯 单相机纠偏-偏心旋转-产品特征标定

生成时间：2026-05-26 16:43  ·  方案设计  ·  定位引导

📋 场景分析

单相机固定安装，向下拍摄机构上的产品。产品在机构上可X/Y/R移动。无标定板时，以产品自身稳定特征（圆孔/Mark点/角点）作为标定参照物。产品与R轴不共轴（偏心），需要输出角度偏差进行纠偏。

关键参数确认值
相机数量与安装方式单相机，固定安装（静止）
应用类型纠偏
机构类型XYR一体
旋转需求有（需输出角度）
工具共轴性不共轴（偏心）
标定条件无标定板 → 使用产品特征点代替
标定方案十二点标定（平移旋转标定）
计算模型单点纠偏（内部加载标定文件）

🗺️ 定位方案思路

选取产品特征点 → 机构移动9位置拍照 → 记录像素+物理坐标 → 旋转中心标定 → 示教基准 → 单点纠偏计算（已加载标定文件） → 输出纠偏坐标→PLC
关键思路：无标定板时，以产品自身稳定特征点作为标定参照物。机构带产品走9个位置建立像素→物理映射关系，再在中心点旋转多角度拟合旋转中心，实现偏心+角度补偿。

⚙️ 操作要点

🔵 标定阶段（十二点平移旋转标定 · 产品特征代替标定板）
选取特征→ 机构走9点→ 旋转3角度→ 标定完成
  1. 1在产品上选取一个稳定的特征点（圆孔优先 > Mark点 > 角点），确保特征边沿清晰、光照稳定、重复性好
  2. 2配置VM图像源和特征提取模块：图像源 → 轮廓匹配(粗定位) → 位置修正 → 圆查找(精定位) + 直线查找(角度查找)
  3. 3配置通信：全局触发建立接收数据+协议解析，PLC每到位一次发送 (C1,x,y,r) 触发标定拍照
  4. 4平移标定：机构带产品在相机视野内走Z字形9个位置（覆盖视野1/3以上），每到位拍照记录像素坐标和物理坐标
  5. 5旋转标定：回到第5点位置，R轴顺/逆时针旋转3次（每次10-30度），XY保持不动，记录特征点像素变化
  6. 6平移旋转标定模块参数：标定类型=平移旋转标定，相机模式=相机静止，自由度=2，平移次数=9，旋转次数=3
  7. 7验证标定结果：状态=1（成功），平移像素误差<1像素，旋转像素误差<0.5像素，角度旋转一致性验证
🟢 示教阶段（建立基准）
基准姿态→ 定位第五点拍照→ 创建基准
  1. 1以基准姿态（角度=0）吸取产品
  2. 2将机构移动到标定第五点拍照位（即标定时的中心点位置），记录此时机构物理坐标为示教物理点
  3. 3拍照获取产品特征点像素坐标，作为基准像素坐标
  4. 4在单点纠偏模块中配置：填写示教物理点、订阅基准图像像素点、执行单次计算 → "创建基准"
🟠 生产阶段（纠偏计算）
拍照位拍照→ 提取特征点→ 单点纠偏→ 输出坐标→PLC
  1. 1机构带产品移动到拍照位（必须与示教拍照位一致）
  2. 2相机拍照，特征提取模块获取当前产品特征点的像素坐标+角度
  3. 3单点纠偏模块加载标定文件，将基准像素坐标和当前像素坐标均转换到物理坐标系，计算偏差（ΔX, ΔY, Δθ）
  4. 4模块输出绝对纠偏坐标（或相对偏差），通过格式化+发送数据送PLC
  5. 5PLC收到坐标后驱动机构纠偏到目标位

⚠️ 注意事项

产品特征点必须稳定可重复提取——建议在光照稳定的环境下测试特征提取的重复性（30次静态测试，像素波动≤1像素）。如果产品表面反光或特征不稳定，需增加光源或更换特征。
标定点位须覆盖相机视野的1/3以上区域，否则外推区域误差会急剧增大。建议Z字形走点，视野中心区域可适当加密。
无标定板时标定精度受限于产品特征点提取精度。如果产品特征本身精度有限（如铸造毛边、冲压塌边），建议考虑增加辅助标定片或打标点来提升精度。
