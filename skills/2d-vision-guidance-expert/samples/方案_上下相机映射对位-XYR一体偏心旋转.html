📋 场景分析

上下双相机均固定安装，XYR一体机构，工具偏心（不共轴），有标定板，需输出XYR绝对坐标的对位贴合场景。

机构类型XYR一体
旋转需求需输出角度偏差
工具情况偏心（不共轴）
标定条件有标定板
标定方案下相机十二点标定 + 上下相机映射标定
生产计算模型单点映射对位

🗺️ 定位方案思路

下相机十二点标定 → 上下相机映射标定 → 示教基准点 → 单点映射对位计算 → 输出XYR绝对坐标
💡 下相机十二点标定：因偏心+有旋转，需标定旋转中心。上下相机映射标定通过标定板统一两相机坐标系。示教记录贴合零位物理坐标。生产时综合上下相机特征输出XYR绝对坐标给机构。

⚙️ 操作要点

标定阶段

下相机平移旋转标定（十二点）

  1. 1机械手吸取标定板移动至下相机视野内，调整至清晰对焦
  2. 2配置平移旋转标定模块：相机静止模式、标定类型选"平移旋转标定"（因不共轴偏心）、平移9次+旋转3次
  3. 3机械手按Z字路径平移9次，每到位一次PLC发送物理坐标触发拍照，保持R轴角度固定
  4. 4回到第5点，R轴旋转3个角度（如-10°、0°、+10°），XY坐标保持不动
  5. 5检查标定结果：标定状态=1，平移像素误差<1px，旋转像素误差<0.5px，角度旋转一致性正确
  6. 6保存下相机标定文件，供生产时单点映射对位模块加载

上下相机映射标定

  1. 7将标定板放在上相机视野内（特征面朝上），上相机拍照采集图像特征
  2. 8机械手吸取标定板（记录此时物理坐标作为示教点），只平移不旋转地移动到下相机视野内
  3. 9下相机拍照采集标定板图像特征
  4. 10搭建相机映射流程：左右分别为上下相机采集的对应特征点，使用"相机映射"模块计算映射矩阵
  5. 11保存映射标定文件，供生产时单点映射对位模块加载
示教阶段
  1. 1记录映射标定第8步中机械手吸取标定板时的物理坐标（称为"示教物理点"）
  2. 2将该物理坐标填入单点映射对位模块的"示教物理点"参数
生产阶段
  1. 1机械手吸取物料移动至下相机拍照位（与标定时下相机拍照位一致），触发下相机拍照
  2. 2下相机流程提取物料对象特征点（圆+角度），通过全局变量传递给上相机流程
  3. 3机械手移动至上相机拍照位（与标定时上相机拍照位一致），触发上相机拍照
  4. 4上相机流程提取目标特征点（目标位置+角度），传入单点映射对位模块
  5. 5单点映射对位模块内部加载下相机标定文件+映射标定文件，输出XYR绝对坐标
  6. 6格式化模块订阅坐标结果，通过发送数据模块将XYR发送给PLC，机构执行对位

⚠️ 注意事项

⚠️ 映射标定过程中，标定板从下相机位移动到上相机位时，机构R轴只能平移不能旋转。若发生旋转，映射关系将被破坏，需重新标定。
⚠️ 生产时上下相机的拍照位必须与标定时保持一致（同一位姿、同一工作距离）。拍照位变化会直接破坏映射关系，导致对位失败。
⚠️ 单点映射对位模块需要加载两个标定文件：下相机平移旋转标定文件 + 上下相机映射标定文件。同时需正确配置示教物理点，否则输出坐标会偏。
