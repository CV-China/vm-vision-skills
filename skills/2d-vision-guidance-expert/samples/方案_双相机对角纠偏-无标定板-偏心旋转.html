🎯 双相机对角纠偏-无标定板-偏心旋转

生成时间：2026-05-26 21:02  ·  方案设计  ·  定位引导

📋 场景分析

双相机静止安装，分别拍摄大物料对角两个特征点。机构为XYR一体，工具偏心安装。无标定板时，两相机各自与机构做十二点标定（产品特征点代替标定板），两相机通过机构物理坐标系统统一，实现对物料对角的XY+角度纠偏。

关键参数确认值
相机数量与安装方式双相机，同侧静止安装（拍对角）
应用类型纠偏（机构带物料，纠正后放置）
机构类型XYR一体
旋转需求有（需输出角度偏差）
工具共轴性不共轴（偏心）
标定条件无标定板 → 产品特征点代替
标定方案双十二点标定（两相机各自做平移旋转标定）
计算模型单点纠偏（内部加载主相机标定文件）

🗺️ 定位方案思路

主相机十二点标定（像素↔物理） + 辅相机十二点标定（像素↔物理） → 两相机拍对角特征点 → 标定转换（各自像素→机构物理） → 点点测量（求中点+角度） → 标定转换（机构物理→主相机像素） → 单点纠偏计算（已加载标定文件） → 输出纠偏坐标→PLC
关键思路：无标定板时，两相机各自通过产品特征点与机构做十二点标定，建立独立的像素→物理转换关系。两相机共享同一个机构物理坐标系，即可在物理空间中直接计算对角中点和连线角度，再转回主相机像素坐标输入单点纠偏模块。注意两相机第五点拍照位偏移量需补偿。

⚙️ 操作要点

标定阶段（双相机各自十二点平移旋转标定 · 产品特征代替标定板）
选产品特征点→ 机构走9平移+3旋转→ 记录像素+物理→ 主相机标定完成 | 辅相机同理→ 辅相机标定完成
  1. 1统一产品特征点：在大物料对角两端各选一个稳定特征点（圆孔优先 > Mark点 > 角点），确保主辅相机分别能拍到各自对角的特征点，边沿清晰、光照稳定
  2. 2搭建主相机标定方案：图像源(主相机) → 轮廓匹配(粗定位) → 位置修正 → 圆查找(精定位) + 直线查找(角度查找) → 平移旋转标定
  3. 3配置通信：添加接收数据和协议解析模块，PLC每到位一次发送 (C1,x,y,r) 触发主相机标定拍照
  4. 4主相机平移标定：机构带产品在主相机视野内走Z字形9个位置（覆盖视野1/3以上），每到位拍照记录像素坐标和物理坐标
  5. 5主相机旋转标定：回到第5点位置，R轴顺/逆时针旋转3次（每次10-30°），XY保持不动，记录特征点像素变化以拟合旋转中心
  6. 6验证主相机标定：平移旋转标定模块参数——标定类型=平移旋转标定，相机模式=相机静止，自由度=2，平移次数=9，旋转次数=3。检查状态=1、平移像素误差<1像素、旋转像素误差<0.5像素
  7. 7辅相机标定：机构带产品移动至辅相机视野内，使用相同的产品特征点重复步骤2-6，完成辅相机十二点标定。两相机共用同一机构物理坐标系
  8. 8记录两相机第五点偏移量：标定完成后，记录主相机第五点的物理坐标 (X5_main, Y5_main) 和辅相机第五点的物理坐标 (X5_aux, Y5_aux)，计算偏移量 ΔX = X5_main - X5_aux, ΔY = Y5_main - Y5_aux，用于后续验证两家标定物理坐标一致性
示教阶段（建立双相机基准姿态）
物料基准姿态→ 两相机同时拍对角→ 求基准中点+角度→ 创建基准
  1. 1以基准姿态（角度=0）吸取物料，移动到拍照位（机构物理坐标为基准拍照位）
  2. 2两相机同时拍摄物料对角，分别提取特征点像素坐标和角度
  3. 3主相机特征点经标定转换到机构物理坐标P_main，辅相机特征点经标定转换到机构物理坐标P_aux
  4. 4点点测量计算中点 P_center = (P_main + P_aux) / 2 和连线角度 θ_ref
  5. 5将中点 P_center 通过标定转换（机构→主相机）转回主相机像素坐标系，得到基准像素坐标
  6. 6记录示教物理点为此时机构物理坐标。在单点纠偏模块中配置：填写示教物理点、订阅基准像素坐标、点击"创建基准"
生产阶段（双相机对角纠偏计算）
移动至拍照位→ 两相机拍对角→ 求中点+角度→ 单点纠偏→ 输出绝对坐标→PLC
  1. 1机构带物料移动到拍照位（须与示教拍照位一致，拍照姿态和工作距离保持不变）
  2. 2两相机同时拍照，分别提取对角特征点的像素坐标和角度
  3. 3主相机特征点经标定转换→机构物理坐标 P'_main，辅相机特征点经标定转换→机构物理坐标 P'_aux
  4. 4点点测量计算当前中点 P'_center 和角度 θ'_line
  5. 5标定转换将 P'_center 从机构物理坐标系转回主相机像素坐标系，得到当前像素坐标
  6. 6单点纠偏模块加载主相机标定文件，将基准像素和当前像素均转换到物理坐标系，计算偏差（ΔX, ΔY, Δθ），输出绝对纠偏坐标
  7. 7格式化模块收集偏差结果，发送数据模块将绝对坐标发送给PLC，引导机构完成纠偏放置

⚠️ 注意事项

产品特征点稳定性是关键：无标定板时标定精度完全依赖产品特征点提取精度。建议先做30次静态测试验证特征点重复性（像素波动≤1像素）。如果产品表面反光、毛刺或塌边，需增加光源或在产品上增设辅助Mark点。
两相机第五点偏移量必须补偿：两相机各自做十二点标定时，第五点（旋转中心位置）的物理坐标不同。生产时机构只能移动到一个拍照位，两相机各拍对角，此时两个特征点在物理坐标系中的坐标差即为两相机第五点偏移量加上物料实际姿态偏移。需确保两相机标定转换后的物理坐标在统一坐标系下一致。
机构拍照位须与示教位一致：单点纠偏模块的基准建立在示教拍照位处，生产时拍照位必须与示教时完全一致（包括XY坐标和R轴角度）。如果因物料来料位置变化需要调整拍照位，则拍照位变化量需补偿到最终输出中。
