🎯 上下双静止相机各自基准对位 — XYR一体·有旋转·不共轴·无标定板

生成时间：2026-05-27 17:30  ·  方案设计  ·  定位引导

📋 场景分析

上下两个静止相机的对位贴合场景：上相机静止安装于上相机位拍料盘（目标），下相机静止倒装于下相机位拍物料（对象）。机构为 XYR 一体，工具偏心（不共轴），需输出 R 轴角度偏差，现场无标定板。采用"各自基准"方式：两相机分别与机构做十二点标定（用稳定特征件替代标定板），各自建立像素→机构物理坐标的独立转换关系；生产时由下相机计算物料偏差，上相机提取料盘目标基准，通过旋转/变量计算将偏差作用到目标基准上，最后用单点抓取模块输出对位 XYR 坐标。

关键参数确认值
相机配置上相机（静止安装，上相机位）+ 下相机（静止安装，倒装固定）
应用类型上下对位贴合（料盘 + 物料）
机构类型XYR一体
旋转需求需输出角度偏差（R轴旋转补偿）
工具情况偏心（不共轴，工具中心 ≠ R轴旋转中心）
标定条件无标定板（使用物料/标准件特征点替代）
标定方案上相机十二点标定 + 下相机十二点标定（两相机各自与机构独立标定）
生产计算模型下相机单点纠偏 + 旋转计算 + 变量计算 + 上相机特征 → 单点抓取

🗺️ 定位方案思路

上相机十二点标定
静止·建立像素→物理→下相机十二点标定
静止·建立像素→物理→下相机示教物料基准
单点纠偏创建基准→上相机示教目标基准
单点抓取创建基准→下相机单点纠偏
物料相对基准的XYR偏差→旋转/变量计算
将物料偏差作用到目标基准→单点抓取
已加载上相机标定文件→输出对位XYR→PLC
💡 各自基准的核心思想：上下相机各自独立做十二点标定到机构物理坐标系（原点为各自第五点拍照位），生产时下相机计算物料相对基准的偏差，上相机仅提取目标基准特征（不做纠偏计算）。通过旋转计算把"物料偏差"作用到"目标基准"上，最后用单点抓取模块输出最终对位坐标。两相机之间不需要映射关系——这是无标定板场景下最简洁的双相机对位方案。

⚙️ 操作要点

标定阶段

通用准备（两相机共用）

  1. 1选定一个稳定的标定工件：表面有圆孔 + 长直边的标准件，或当批物料中挑出特征清晰的一块作为标定件（圆孔做精定位，直边做角度查找）
  2. 2静态测试验证：机构静止，连续拍 30 张同一画面，圆心像素波动应 ≤ 1 像素；机构带件平移 0.1mm，像素偏差应可分辨。不达标先优化光源/曝光/对焦再开始标定

上相机平移旋转标定（静止相机·十二点）

  1. 3机构吸取标定工件，举到上相机视野中央对焦清晰，作为上相机拍照位（记录此机构位姿为"上拍照位"）
  2. 4配置平移旋转标定模块：相机静止模式、标定类型"平移旋转标定"、平移9次 + 旋转3次
  3. 5机构带着工件按"Z字形"走9个平移点（R 角保持固定），每到位 PLC 触发上相机拍照，并把机构物理坐标发给视觉。平移步长覆盖视野的 1/3 ~ 1/2
  4. 6机构回到第5点（旋转标定原点），R 轴依次旋转 3 个角度（推荐 -10° / 0° / +10°），XY 保持不动，每次旋转后触发拍照
  5. 7检查上相机标定结果：标定状态=1，平移像素误差 < 1px，旋转像素误差 < 0.5px，XY 轴夹角 89.8°~90.2°
  6. 8保存上相机标定文件（生产时由"单点抓取"模块加载）

下相机平移旋转标定（静止相机·十二点）

  1. 9机构吸取同一标定工件，移到下相机视野中央对焦清晰（下相机倒装，工件朝向下相机），作为下相机拍照位（记录此机构位姿为"下拍照位"）
  2. 10配置平移旋转标定模块：相机静止模式、十二点参数与上相机一致
  3. 11同样"Z字9点平移 + 第5点3角度旋转"，PLC 每点触发拍照并发送机构物理坐标
  4. 12检查下相机标定结果（同步骤7），保存下相机标定文件（生产时由"单点纠偏"模块加载）
示教阶段
  1. 1下相机示教物料基准：机构以基准姿态吸取一块"标准良品物料"，移到下相机拍照位（必须与下相机标定第五点位姿一致），触发拍照。提取物料的圆心像素 + 直边角度，记录为"下相机基准像素点 + 基准角度"，同时记录机构物理坐标为"下相机示教物理点"
  2. 2打开下相机的单点纠偏模块，填入：下相机示教物理点 + 下相机基准像素点，加载下相机标定文件，执行"创建基准"。验证：把基准像素点输入模块，输出偏差应为 (0,0,0)
  3. 3上相机示教目标基准：保持机构基准姿态，移到上相机拍照位（必须与上相机标定第五点位姿一致），触发上相机拍料盘（目标）。提取料盘的圆心像素 + 直边角度，记录为"上相机目标基准像素 + 基准角度"，同时记录机构物理坐标
  4. 4打开生产流程中的单点抓取模块，把"上相机目标基准像素 + 基准角度 + 机构物理点"作为基准填入，加载上相机标定文件，执行"创建基准"
  5. 5对位试运行验证：把基准物料 + 基准料盘以基准姿态放好，跑一次完整流程，输出对位 XYR 应为 (0,0,0)，机构执行不动作。这是判断示教是否正确的金标准
生产阶段

计算模型：下相机单点纠偏 + 旋转计算 + 变量计算 + 单点抓取

  1. 1机构吸取来料物料，移到下相机拍照位（与下相机示教位姿完全一致），触发下相机拍照
  2. 2下相机视觉流程：图像源 → 轮廓匹配（粗定位）→ 位置修正 → 圆查找（精定位）+ 直线查找（角度查找）→ 单点纠偏（输出物料相对下相机基准的 XYR 偏差，记为 ΔX_下、ΔY_下、Δθ）
  3. 3机构保持物料姿态不变，移到上相机拍照位（与上相机示教位姿完全一致），触发上相机拍料盘
  4. 4上相机视觉流程：图像源 → 轮廓匹配 → 位置修正 → 圆查找 + 直线查找（仅提取料盘当前的圆心像素 + 角度，不做单点纠偏计算）
  5. 5旋转计算模块：以下相机解出的 Δθ 为旋转角，把上相机提取的目标特征点绕物料旋转中心旋转 Δθ，得到"物料按基准姿态摆正后"目标特征点应该出现的图像位置
  6. 6变量计算模块：合并下相机给出的 (ΔX_下, ΔY_下, Δθ) 与上相机旋转后的目标特征，组装成"目标基准像素 + 当前姿态角度"，作为单点抓取的运行输入
  7. 7单点抓取模块（已加载上相机标定文件）：以基准为参考，计算当前物料贴合目标料盘所需的 XYR 绝对坐标
  8. 8格式化 → 发送数据，把 XYR 三个值发给 PLC，机构动作完成对位贴合

⚠️ 注意事项

⚠️ 两相机拍照位必须固化：上下相机各自的物理坐标系原点是各自标定第五点（拍照位）。生产时机构去上/下相机拍照位的位姿必须与标定时第五点 + 示教时基准位完全一致，任何拍照位漂移都会被直接当成对位偏差输出，造成虚假误差。建议把两个拍照位写死为 PLC 上的两个固定示教点。
⚠️ 无标定板对特征件的要求：用物料/标准件代替标定板时，特征必须稳定可重复（圆孔 + 长直边组合最佳）。开标前先做静态/动态重复性测试：静态像素波动 ≤ 1 px，动态平移重复性 ≤ 0.02 mm。特征不稳定会直接破坏十二点标定结果——这是无标定板方案的最大风险点。
⚠️ 不共轴 + 有旋转 → 必须十二点，且第五点位姿要严格复现：偏心场景下 R 轴旋转会引入 XY 平移耦合（旋转中心 ≠ 工具中心）。十二点标定用 9 平移 + 3 旋转拟合出旋转中心。生产中机构每次走到"第五点拍照位"必须精度一致（推荐重复定位精度 ≤ 0.01 mm），否则角度越大、对位偏差越大。
