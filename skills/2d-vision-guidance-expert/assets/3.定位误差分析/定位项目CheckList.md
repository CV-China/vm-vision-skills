## 1. 视觉硬件

### 1.1 光源

| 检查项 | 检查内容 | 备注 |
|-------|---------|------|
| **光源控制** | 光源控制可靠，打光效果能满足要求 | 如果不能，立即更换相机和光源，不要心存侥幸 |
| **吸嘴影响** | 特别注意吸嘴是否会影响成像 | - |
| **打光方案** | 现场选取的抓边等基准一直在变，现场需要多种光源及专业的打光工程师 | - |

**检查清单**：
- [ ] 光源控制可靠
- [ ] 打光效果满足要求
- [ ] 吸嘴不影响成像
- [ ] 准备多种光源方案
- [ ] 配备专业打光工程师

### 1.2 相机

| 检查项 | 检查内容 | 备注 |
|-------|---------|------|
| **相机选型** | 对于定位类项目，一定要选用成像质量较好的相机 | 就算分辨率再高，对精度也没有提升 |
| **视野范围** | 被测物体不要完全充满视野 | 最好在三分之二范围内，太小太大都会引入误差 |
| **安装检查** | 验证相机的垂直度，松紧，以及安装轴的垂直度 | - |

**检查清单**：
- [ ] 选用成像质量好的相机
- [ ] 被测物体占视野2/3范围
- [ ] 相机安装垂直
- [ ] 相机固定牢固
- [ ] 安装轴垂直

## 2. 结构硬件

### 2.1 重复精度

| 检查项 | 检查内容 | 备注/目标 |
|-------|---------|----------|
| **静态精度** | 结构不移动，相机连续拍摄同一个对象，输出静态精度报告 | - |
| **动态精度** | 结构往复移动，相机定位同一个对象，输出动态精度报告 | 特别注意R轴的精度验证 |
| **精度证明** | - | 证明机构精度没问题 |

**检查清单**：
- [ ] 完成静态精度测试
- [ ] 完成动态精度测试
- [ ] R轴精度验证通过
- [ ] 输出静态精度报告
- [ ] 输出动态精度报告

### 2.2 硬件设计

| 检查项 | 检查内容 |
|-------|---------|
| **垂直度检查** | 通过千分表（平面度/垂直度）和印泥等手段确认结构是否垂直 |
| **水平度检查** | 通过千分表验证相机、镜头和吸嘴是否水平 |
| **平面度检查** | 验证组装时，组装平面的平行度，下压是否导致贴合物体发生偏移 |
| **极限位置** | 运动机构不能够在极限位置进行工作，前期尽可能要求修改 |
| **工具坐标系** | 对于旋转臂较长的单位必须建立工具坐标系，防止大悬臂导致的旋转误差 |

**检查清单**：
- [ ] 结构垂直度检查通过
- [ ] 相机水平
- [ ] 镜头水平
- [ ] 吸嘴水平
- [ ] 组装平面平行度检查通过
- [ ] 避免极限位置工作
- [ ] 旋转臂较长时建立工具坐标系

### 2.3 运动控制

| 检查项 | 检查内容 |
|-------|---------|
| **到位确认** | 拍照时一定要注意机构是否到位 |
| **延时设置** | 拍照后一定要加延时 |
| **离开方式** | 离开物体表面时一定要先垂直离开，不得直接回到安装工位 |

**检查清单**：
- [ ] 确认机构到位后再拍照
- [ ] 拍照后加适当延时
- [ ] 垂直离开物体表面
- [ ] 避免直接回到安装工位

## 4. 标定

### 4.1 畸变矫正

| 检查项 | 检查内容 |
|-------|---------|
| **畸变评估** | 目前的硬件选型使得产品基本充满视野了，建议全部先做畸变矫正 |
| **矫正必要性** | 根据畸变系数确认是否需要做畸变矫正 |

**检查清单**：
- [ ] 评估畸变情况
- [ ] 计算畸变系数
- [ ] 确定是否需要畸变矫正
- [ ] 完成畸变矫正（如需要）

### 4.2 N点标定

| 检查项 | 检查内容 |
|-------|---------|
| **标定范围** | 标定范围应尽可能大（2/3视野） |
| **数据保存** | 将每次标定信息存下来，包括图像及点位数据，便于后续分析 |
| **点位精度** | 保证点位提取精度（1个像素以内波动） |
| **角度精度** | 保证抓边角度精度（波动0.1度以内） |
| **等间距检查** | 特别注意标定的间距是否是等间距 |
| **路径检查** | 标定的路径是否是直线 |
| **精度一致性** | 标定的X精度与Y精度是否相同 |
| **重合点检查** | 标定的第五点和第11点是否重合 |
| **脉冲当量** | 确认客户设置的脉冲当量正确 |

**检查清单**：
- [ ] 标定范围覆盖2/3视野
- [ ] 保存标定图像
- [ ] 保存标定点位数据
- [ ] 点位提取精度在1个像素以内
- [ ] 抓边角度精度波动在0.1度以内
- [ ] 标定间距等间距
- [ ] 标定路径为直线
- [ ] X精度与Y精度相同
- [ ] 第五点和第11点重合
- [ ] 脉冲当量设置正确

### 4.3 映射标定

| 检查项 | 检查内容 |
|-------|---------|
| **点对应关系** | 对位映射一定要确保上下相机图像四个点对应正确 |
| **标定方法** | 可以上下图像四个点分别按照顺时针、逆时针的方式标序号一一对应 |

**检查清单**：
- [ ] 确认上下相机图像四个点对应正确
- [ ] 按照顺时针/逆时针方式标序号
- [ ] 完成一一对应标定

## 5. 定位粗调

| 检查项 | 检查内容 | 备注/目标 |
|-------|---------|----------|
| **中心对中心** | 创造条件，验证大角度大位移下，中心对中心可以稳定贴合 | - |
| **偏移值方式** | 创造条件，验证大角度大位移下，偏移值的方式可以稳定贴合 | 证明计算方法没问题 |

**检查清单**：
- [ ] 大角度大位移下中心对中心验证通过
- [ ] 大角度大位移下偏移值方式验证通过
- [ ] 证明计算方法正确

## 6. 定位精调

### 6.1 补偿值

| 检查项 | 检查内容 |
|-------|---------|
| **补偿频率** | 不得频繁调整补偿值，务必保证一致性很高的情况下做补偿 |
| **补偿值限制** | 补偿值不得大于0.1，如果大于0.1，必须搞明白在补偿什么东西 |

**检查清单**：
- [ ] 一致性很高时才做补偿
- [ ] 补偿值不大于0.1
- [ ] 明确补偿内容

### 6.2 抓边稳定性

| 检查项 | 检查内容 |
|-------|---------|
| **大量测试** | 累计保存大量物料图片进行测试 |
| **稳定性验证** | 保证抓边及特征点稳定 |
| **参数修改** | 如修改参数需重新测试 |

**检查清单**：
- [ ] 累计保存大量物料图片
- [ ] 抓边稳定
- [ ] 特征点稳定
- [ ] 修改参数后重新测试

---

## 7. 复检

| 检查项 | 检查内容 |
|-------|---------|
| **光照一致** | 复检时尽量保证在一张图片内测量 |
| **光源一致性** | 如不可以在一张图片内测量，尽量保证光源一致性 |

**检查清单**：
- [ ] 尽量在一张图片内测量
- [ ] 保证光源一致性

---

## 8. VM方案

| 检查项 | 检查内容 |
|-------|---------|
| **条件检测** | 格式化前面必须接条件检测，确保前置关键模块都OK的情况下才能输出OK |
| **模板修正** | 修改模板后一定记得固定位置修正的基准点，以保证后续使用位置修正信息模块的ROI正确 |
| **输出信息** | 定位项目除输出X,Y,R及抓边特征渲染图，便于后期排查 |

**检查清单**：
- [ ] 格式化前接条件检测
- [ ] 关键模块OK才输出
- [ ] 修改模板后固定基准点
- [ ] 位置修正信息模块ROI正确
- [ ] 输出X,Y,R坐标
- [ ] 输出抓边特征渲染图
