## 1. 标定与定位概述

### 1.1 定位应用三部曲

无论你是新手还是老手，无论您是否理解，请先记住这三个步骤：

#### ① 标定

**标定的目的**：获取相机内外参矩阵，计算从相机的像素坐标系到现实的世界坐标系的映射关系。

```
图像坐标 (U, V) → 标定矩阵 → 物理坐标 (X, Y)
```

#### ② 示教

**示教的目的**：创建基准。

- **示教姿态**：工具在标准位置时的状态
- **运行姿态**：工具在实际运行时的状态
- **基准**：工具在运行时需要到达的标准位置

#### ③ 做差

**总体偏差 = 旋转偏差 + 平移产生的偏差**

所有定位应用的计算流程归根到底就是**偏差计算**，理解怎么计算偏差就掌握了所有定位应用的计算方法。

- 旋转矩阵：描述旋转关系
- 旋转中心：旋转的参考点
- 平移偏差：位置偏移量
- 旋转偏差：角度偏移量

---

### 1.2 共轴与不共轴

根据吸嘴与旋转轴是否共轴，可以将定位场景分为两大类：

| 类型 | 特点 | 标定方式 |
|-----|------|---------|
| **共轴** | 吸嘴与旋转轴重合 | 平移标定（9点标定） |
| **不共轴** | 吸嘴与旋转轴分离 | 平移旋转标定（9+3点标定） |

---

### 1.3 广义标定

**观察有几个坐标系，思考如何统一到一个坐标系。**

> 只有统一到一个坐标系下计算才不会乱！

这个过程就是标定，只不过从图像到物理一般叫做"标定"，而把图像到图像叫"映射"，两者本质是一样的。

## 2. 标定与定位模块

### 2.1 模块概述

相比VM4.0，VM4.2新增定位工具箱，内含优化后的标定及定位模块，代替以往脚本代码来计算偏移量的方法，零代码落地常规定位应用。

#### 标定模块

| 模块 | 功能 | 适用场景 |
|-----|------|---------|
| **平移旋转标定** | 支持平移标定和平移旋转标定 | 共轴/不共轴场景 |
| **旋转标定** | 计算机构旋转中心像素坐标与物理坐标 | 小角度标定场景精度更高 |

#### 定位模块

| 模块 | 功能 | 适用场景 |
|-----|------|---------|
| **单点抓取** | 基于图像基准点与图像运行点坐标实现抓取 | 单相机拍物料的抓取场景 |
| **单点纠偏** | 基于图像基准点与图像运行点坐标实现纠偏 | 单相机拍物料的纠偏场景 |
| **单点映射对位** | 基于对象图像特征点与目标图像特征点坐标实现上下相机对位 | 上相机拍目标，下相机拍对象的对位贴合场景 |

---

### 2.2 平移旋转标定模块

**适用场景**：单相机与机构（模组/机械手）的标定场景。

> **物理坐标支持订阅和手动输入两种！**

#### 模块参数

| 参数 | 选项 | 说明 |
|-----|------|------|
| **标定类型** | 1.平移标定（不计算旋转中心）<br>2.平移旋转标定（计算旋转中心） | 根据共轴/不共轴选择 |
| **标定点获取** | 1.触发获取<br>2.手动获取 | 数据采集方式 |
| **相机模式** | 1.相机运动<br>2.相机静止 | 相机安装方式 |
| **自由度** | 1.缩放、旋转、纵横比、倾斜、平移、透射<br>2.缩放、旋转、纵横比、倾斜、平移（默认）<br>3.缩放、旋转、平移 | 变换自由度选择 |
| **标定点输入** | 按点、按坐标 | 输入方式 |
| **平移次数** | 默认9次 | 平移标定点数 |
| **旋转次数** | 默认3次（平移标定为0不显示） | 旋转标定点数 |
| **组合标定** | 开启/关闭 | 支持小视野旋转后平移回视野内的标定 |
| **更新文件** | 指定路径 | 标定文件保存位置 |
| **示教** | 传入示教物理值 | 将当前图像特征点作为基准 |

#### 平移标定

**适用场景**：
1. 机械手建立工具坐标系，机械的抓取点所在位置即是机械手当前坐标
2. 抓取产品时，如果需要纠正被抓产品角度，旋转是绕着抓取点旋转

> 不带旋转的N点标定适用场景很有限，但对视觉来说非常简单，目标点直接标定转换即可，无需关注角度变化带来的影响。

#### 平移旋转标定

**适用场景**：
1. 机械手无需建立工具坐标系，机械的旋转中心所在位置即是机械手当前坐标
2. 抓取产品时，如果需要纠正被抓产品角度，旋转是绕着旋转中心旋转

> 带旋转的N点标定适用绝大多数应用场合，图像点直接标定转换的数值不能直接用，需计算角度变化带来的影响，且旋转半径越大定位误差带来的影响越大。

#### 标定的细节

- **平移旋转标定的顺序**：约定走S形，但不是必须；如果是按照通讯协议标定，机械运行轨迹任意都可以
- **旋转中心**：拟合旋转中心得到的像素点，标定转换后的坐标点为（0,0）

### 2.3 旋转标定模块

**适用场景**：单相机与机构（模组/机械手）的旋转中心标定场景。

> **物理坐标支持订阅和手动输入两种！**

#### 模块参数

| 参数 | 说明 |
|-----|------|
| **输入方式** | 支持按点、按坐标两种方式 |
| **图像点** | 图像中特征点的像素坐标 |
| **物理旋转角度** | 机构每次旋转的角度值 |
| **旋转次数** | 机构旋转的次数（最少为三次） |

### 2.4 单点抓取模块

**适用场景**：单相机拍物料的抓取场景。

> **像素点使用了点位复用，需要注意创建基准步骤！**

**限制条件**：不支持多相机联立坐标系、分离轴抓取、相机拍照位变化（可后接变量计算模块实现）等场景。

#### 模块参数

| 参数 | 说明 |
|-----|------|
| **标定类型** | 1.平移标定（不计算旋转偏差）<br>2.平移旋转标定（计算旋转偏差） |
| **像素点** | 1.当前图像特征点（X,Y）<br>2.当前物料的角度 |
| **示教抓取物理点** | 1.物理点（X,Y）<br>2.物理角度 |
| **示教拍照物理点** | 1.拍照位物理点（X,Y）<br>2.拍照物理角度 |
| **标定文件** | 支持xml/iwcal格式标定文件 |
| **刷新信号** | 填1会自动加载最新的标定文件 |
| **创建基准** | 记录基准位图像像素坐标及示教物理点坐标 |

> **注意**：该模块默认拍照位为固定拍照位，相机移动拍照位的场景需要接变量计算模块。

## 3. 典型定位应用场景

### 3.1 单点抓取典型场景

#### 场景说明

**应用场景**：物料的抓取应用，算法需要返回给机构相对偏移坐标或绝对抓取坐标。

**机构限制**：机构是XYR一体，相机安装在R轴上，可随XY一起平移，随R轴一起旋转；或相机静止。

#### 标定步骤

1. **选择标定方式和相机状态**
2. **标定动作**：
   - 在视野中放置一个物料或标定片
   - 机构带着相机平移9次，平移中保持角度固定
   - 回到中心点，机构带着相机旋转3次，旋转中XY保持固定

#### 示教步骤

1. 放置生产用的物料在工作台中
2. 调整机构位置，使得机构的吸嘴或夹爪可以很好的对准物料，记录此时机构的物理坐标
3. 机构移动到拍照位置（即标定中的第五点坐标），拍摄物料，获得像素坐标
4. **创建基准**：
   - 示教抓取物理点：示教时抓取物料的机械手物理坐标
   - 基准图像像素点：此时拍照位拍照获取的图像像素点

#### 生产注意事项

1. 生产拍照位与标定时的工作距离必须保持不变
2. 生产拍照位坐标最好与标定第五点拍照位一致，可产生X/Y偏移，但拍照姿态（相机旋转角度）必须保持不变
3. 生产时需保证提取的特征点与示教图像特征点一致
4. 示教物理点为示教时刚好抓取到物料时的机械手坐标

---

### 3.2 单点纠偏典型场景

#### 场景说明

**应用场景**：物料的纠偏应用，算法需要返回给机构相对偏移坐标或绝对纠偏坐标。

**机构限制**：机构是XYR一体，相机静止下相机位。

#### 标定步骤

1. **选择标定方式和相机状态**
2. **标定动作**：
   - 机械手吸取一个物料或标定片在视野中
   - 机构带着物料平移9次，平移中保持角度固定
   - 回到中心点，机构带着物料旋转3次，旋转中XY保持固定

#### 示教步骤

1. 以标准姿态抓住物料吸嘴或者夹爪中
2. 记录此时机构的物理坐标
3. 拍摄物料，获得像素坐标
4. **创建基准**：
   - 此时机构的物理坐标为示教物理点
   - 获得的图像坐标按如图所示创建基准

#### 生产注意事项

1. 生产时机械手抓取物料移动到示教拍照位
2. 生产时需保证提取的特征点与示教图像特征点一致

---

### 3.3 上下相机映射对位典型场景

#### 场景说明

**应用场景**：上相机拍目标，下相机拍对象的对位贴合场景。

**机构限制**：有两个相机，目标相机（一般是上相机）和对象相机（一般是下相机），目标相机安装在执行机构上或静止安装，对象相机倒装。

> **关键**：统一到下相机图像坐标系求偏差是关键

#### 标定步骤

**标定选择**：选择标定方式和相机状态；上下相机映射标定，下相机平移旋转标定。

**标定动作**：

1. **映射标定**：
   - 上相机拍摄标定板（海康Ⅰ型/Ⅱ型标定板）
   - 移动机械手抓取标定板并移动至下相机位
   - 下相机拍照
   - 将两张标定板图像输入映射标定模块，生成映射标定文件

2. **下相机平移旋转标定**：
   - 同单点抓取/纠偏的标定方法

#### 示教步骤

**示教物理点**：映射标定时机构反向吸取标定板时的绝对物理坐标。

#### 生产步骤

1. 首先机械手抓取物料移动至下相机拍照位获取物料的特征点
2. 然后机械手移动至上相机拍照位，拍摄待贴合物料，进行偏差计算

---

### 3.4 定位应用类型总结

复杂的定位应用就是上面4种基本类型的各种组合：

| 基本类型 | 特点 |
|---------|------|
| **单点抓取** | 单相机，物料在固定位置，机械手抓取 |
| **单点纠偏** | 单相机，物料在机械手上，需要纠偏 |
| **上下相机对位** | 双相机，上相机拍目标，下相机拍对象 |
| **组合应用** | 上述类型的组合 |

## 4. 注意事项与问题排查

### 常见定位机构问题排查思路

#### 问题1：拍照位角度姿态不一致

**现象**：定位拍照时发现拍照位角度姿态没有与标定时保持一致，导致定位不准。

**排查方式**：比对标定时的拍照姿态与生产时的标定姿态，二者常规情况下角度必须一致。

#### 问题2：机构未走到对应位置

**现象**：机构拍照未走到对应位置，导致定位不准。

**排查方式**：比对同一点位的生产图片，观察背景是否偏移。

#### 问题3：运动机构不稳定

**现象**：运动机构不稳定，导致定位不准确。

**排查方式**：可以对每个相机进行动静态测试。

| 测试类型 | 方法 | 判定标准 |
|---------|------|---------|
| **静态测试** | 保持拍照位不动，连续100次拍摄同特征，输出excel结果，计算波段峰峰值 | 1个像素以内为正常 |
| **动态测试** | 机构移动至拍照位拍照，连续往返100次拍摄同特征，输出excel结果，计算波段峰峰值 | 机构运动误差小于0.02mm为正常 |

- 静态结果小，表示视觉稳定性好；反之，则视觉稳定性差
- 动态结果小，表示视觉及机构整体的稳定性好；反之，则整体稳定性差

#### 问题4：吸嘴与平台不水平

**现象**：吸嘴末端与物料平台不水平，吸放物体时可能发生侧压，复检时发现引导误差偏大。

**排查方式**：通过水平尺或者其他测试水平的工具，测试吸嘴平面与目标平面是否平行。

#### 问题5：大角度偏移不稳定

**现象**：当机构各项测试稳定，涉及小角度偏移时稳定，大角度偏移时不稳定。

**排查方式**：检查机械手各关节位置是否产生了限制，尽量让关节不受限，测试有无改善。
