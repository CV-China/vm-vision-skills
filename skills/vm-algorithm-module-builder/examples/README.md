# Examples 索引

本目录收录**脱敏后**的算法模块场景示例，覆盖典型输入/输出形态组合。每个示例包含**关键代码片段 + XML 节点**，**不包含业务算法逻辑**（一律以 `// === 算法核心处理（占位） ===` 注释代替）。

> ⚠️ 注意：示例已**移除**所有客户业务字段名，统一改为 `Output1` / `Output2`。如用户需要按业务场景命名，由 skill 在生成时根据用户需求重命名。

| 编号 | 场景 | 输入 | 输出 | 关键点 |
|---|---|---|---|---|
| 03 | 多图像输入 | 2 张图（主图 + 参考图） | 1 张图 + 距离值 | VmModule_GetInputImageEx 多图获取 |
| 05 | OpenCV 集成 | 1 张图 | 1 张图（Canny 边缘） | HKA_IMAGE ↔ cv::Mat 互转 |
| 06 | HALCON 集成 | 1 张图（灰度） | 1 张图（缩放后） | IMvdImage ↔ HObject 互转、GenImage1Extern、HException 处理 |

## 复用方式

skill 在 §plan-before-code 阶段，确认用户场景与哪个示例相近，列出"参考示例 0X"，让用户对最终代码组合有预期。**不需要**整工程级复制示例代码——AlgTemplate 模板才是基础。

## 为什么这里不放完整工程？

- 完整工程 5MB+，全部冗余复制会臃肿
- 示例的价值在**形态特征**（哪些 XML 节点、哪些 C++ 接口组合），不在算法
- 生成时按 §澄清结果，从 io-params/ 和 xml-schemas/ 拼装即可
